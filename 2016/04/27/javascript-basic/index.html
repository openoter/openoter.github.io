JavaScript基础

  • JavaScript高级程序设计
  • 基本概念

1. 数据类型

在JavaScript中包含5中基本数据类型和一种复杂数据类型。
基本数据类型：Undefined、Null、Boolean、Number、String
复杂数据类型：Object

  • typeof：用来检测给定变量的数据类型
  • undefined：该类型只有一个值（undefined），一个变量申明了但是没有赋值时，它默认的值就为undefined
  • Null类型：表格一个空对象指针，使用typeof检测时，返回的是Object

如果定义的变量准备用于保存对象，最好将该变量初始化为null,而不是其他的值。

  • Number类型
    • 浮点数值
    • 数值范围
    • NaN
    • 数值转换
  • String类型
    • 字符字面量
    • 字符串的特点
    • 装换为字符串（toString()）

2. Object类型

在创建对象时，建议把括号带上。

Object的每个实例都有以下属性和方法。

  • constructor：保存着用于创建当前对象的函数。

  • hasOwnProperty：用于检查给定的属性在当前对象实例中是否存在。属性名必须以字符串形式给定。

  • isPrototypeOf(Object):用于检查当前对象是否是当前对象的原型。

  • prototypeIsEnumerable(propertyName):用于检查给定的属性是否能够使用for-in语句来枚举。
  • toLocalString()：返回对象的字符串表示（与执行环境的地区对应）
  • toString()：返回对象的字符串表示
  • valueOf()：返回对象的字符串、数值或布尔表示。通常与toString()返回值相同。

3. 操作符

  • 一元操作符（++、–，+、-）
  • 位操作符（~、|、&、^、<<、>>、>>>）
  • 布尔操作符（!、&&、||）
  • 乘性操作符（*、/、%）
  • 加性操作符（+、-）
  • 关系操作符（<、>、<=、>=）

  • 相等操作符（==、!=、===）
    |表达式|值|表达式|值|
    |——|–|—–|–|
    |null == undefined|true|true == 1|true|
    |”NaN”==NaN|false|true==2|false|
    |NaN == NaN|false|undefined==0|false|
    |NaN != NaN|true|null == 0|false|
    |5 == NaN|false|”5” == 5|true|
    |false == 0 | true| | |

  • 赋值操作符（=）

  • 逗号操作符（，）

4. 语句

  • if语句
  • do-while语句
  • while语句
  • for语句
  • for-in语句
  • lable语句

    使用label语句可以在代码中添加label，以便于将来使用。

  • break语句和continue语句
  • with语句
    作用：将代码中的作用域设置到一个特定的对象中。
    目的：为了简化多次编写同一个对象的工作。

注意：严格模式下不允许使用with语句

大量使用with语句会导致性能下降，同时也会给调试代码造成困难，因此在开发大性项目时，不建议使用with语句。

  • switch语句

5. 函数

  • 使用关键字function关键字进行申明
  • 理解return
    • 一个函数可以包含多个return语句
    • return语句也可以不带任何返回值，函数在停止执行后返回undefined

严格模式下对函数的限制：

  • 不能把函数名命名为eval或arguments
  • 不能把参数名命名为eval或arguments
  • 不能出现两个命名参数同名的情况
  1. 理解参数

  • 定义的函数有两个参数，使用时不一定要传递两个参数，可以是两个甚至三个。之所以这样，是因为ECMAScript中的参数内部用一个数组来表示。在函数体内可以通过arguments来访问这个参数数组（arguments[0]….）

    • 通过length属性可以获得有多少个参数传递给了函数

    • arguments对象可以与命名参数一起使用

- 严格模式下的限制：
    + 类似`arguments[0] = 10`无效;
    + 重写arguments的值会导致语法错误
  1. 没有重载
