web前端之表单插件

  • web前端
  • 表单

旋钮输入

官方网址：http://anthonyterrien.com/knob/

jQuery Knob是一款基于canvas的一款旋钮输入插件，它实现可以通过touch、mouse、mousewheel、keyboard events实现值得增加。

范围滑块

官网地址：https://github.com/IonDen/ion.rangeSlider

rangeSlider是一款范围滑块的插件,它支持手机端、浏览器端。我们可以通过它创建更加漂亮的滑块。

演示地址

日期选择器

官网地址：https://github.com/eternicode/bootstrap-datepicker

bootstrap-datepicker是bootstrap中的时间选择器，它支持选择年、月、十年、时间范围等。

时间选择器

  • 官网地址：http://weareoutman.github.io/clockpicker/
  • 官网地址：http://sentsin.com/layui/laydate/

clockpicker，顾名思义，就一款用于选择事件时间（小时、分钟、秒）的插件。

固定格式文本

官网地址：https://github.com/RobinHerbots/jquery.inputmask

比如现在有一个需求，用户需要输入ip地址，而且只能输入ip地址。

切换开关

官网地址：http://abpetkov.github.io/switchery/

文件上传

官网地址：https://github.com/episage/bootstrap-3-pretty-file-upload

拾色器

官网地址： http://mjolnic.github.io/bootstrap-colorpicker

图片裁剪

官网地址：http://fengyuanchen.github.io/cropper/

表单验证

官网地址：https://jqueryvalidation.org/

中文deme：http://www.runoob.com/jquery/jquery-plugin-validate.html

表单向导

官网地址：http://www.jquery-steps.com/

文件上传

  • Web Uploader
  • DropzoneJS
  • 富头像上传编辑器

富文本编辑器

Summernote
Summernote是一个简单的基于Bootstrap的WYSIWYG富文本编辑器
官方文档请参考： http://hackerwins.github.io/summernote/features.html#api

Markdown编辑器
Bootstrap-Markdown是一款优秀的markdown编辑器，提供了实用的API，利用插件进行扩展。
更多示例请访问：http://toopay.github.io/bootstrap-markdown/
GitHub： https://github.com/toopay/bootstrap-markdown

combox 搜索建议插件

这是一个基于 bootstrap 按钮式下拉菜单组件的搜索建议插件.

使用说明：https://github.com/lzwme/bootstrap-suggest-plugin
